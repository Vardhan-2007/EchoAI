# 4️⃣ Core Feature: Eisenhower Matrix

Your system uses a famous productivity framework.

### Eisenhower Matrix

Tasks are classified into **4 quadrants**:

| Quadrant | Meaning                    |
| -------- | -------------------------- |
| Q1       | Urgent + Important         |
| Q2       | Important + Not urgent     |
| Q3       | Urgent + Not important     |
| Q4       | Not urgent + Not important |

Example in code:

```
const QUADS = {
 Q1: {label:"Do First"},
 Q2: {label:"Schedule"},
 Q3: {label:"Delegate"},
 Q4: {label:"Eliminate"}
}
```

Your AI automatically assigns:

```
urgency
importance
quadrant
```

Example task object:

```
{
 title:"Submit report",
 urgency:9,
 importance:9,
 quad:"Q1"
}
```
